# AiQ Athlete Dashboard Starter

This project is a starter template for the AiQ Athlete Dashboard, using React, React Router, Tailwind CSS, and Firebase.

## Features
✅ React + Tailwind CSS setup  
✅ React Router with sample pages  
✅ Sample components (Navbar, DashboardCard, VideoUpload)  
✅ Placeholder Firebase configuration

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```

4. Update your Firebase config in `src/firebaseConfig.js`

## Dependencies
- react
- react-dom
- react-router-dom
- tailwindcss
- firebase

## Notes
Remember to run Tailwind CSS build or use a framework like Create React App with Tailwind integration.
